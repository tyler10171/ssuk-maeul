[A] 1차 업로드 예정 정책 (30건)
==============================================================================

------------------------------------------------------------------------------
A-1. 중앙정부 · 임신 (3건)
------------------------------------------------------------------------------
  1) 임신·출산 진료비 (국민행복카드)
     - 임신 확인 후 진료비 바우처
     - 단태아 100만원 / 다태아 140만원
     - 신청: 정부24 또는 카드사 / 분만 후 2년까지

  2) 청소년 산모 임신·출산 의료비 지원
     - 만 19세 이하 산모 추가 의료비
     - 120만원 (기본 진료비 외 추가)

  3) 임산부 친환경농산물 꾸러미
     - 친환경 식재료 꾸러미 제공
     - 연 48만원 / 농림축산식품부 + 지자체 매칭


------------------------------------------------------------------------------
A-2. 중앙정부 · 출산 (3건)
------------------------------------------------------------------------------
  4) 첫만남이용권
     - 출생아 1인당 일시금 바우처
     - 첫째 200만원 / 둘째 이상 300만원
     - 출생 후 1년 내 신청, 24개월 사용

  5) 산모·신생아 건강관리 지원 (산후도우미)
     - 건강관리사 파견 (5~25일)
     - 중위소득 150% 이하
     - 출산 예정일 40일 전 ~ 출산 후 30일 이내 신청

  6) 행복출산 원스톱 서비스
     - 출생신고와 함께 13종 지원 통합 신청
     - 정부24


------------------------------------------------------------------------------

------------------------------------------------------------------------------
  7) 부모급여
     - 만 0세 월 100만원, 만 1세 월 50만원 (현금)
     - 출생 60일 내 신청 시 출생월부터 소급

  8) 아동수당
     - 만 8세 미만 월 10~12만원 (지역별 차등)
     - 2026~2030 단계적으로 만 13세까지 확대

  9) 양육수당 (가정양육)
     - 어린이집·유치원 미이용 아동 / 가정양육
     - 월 10~20만원 (월령별)

 10) 어린이집 보육료
     - 만 0~5세 어린이집 이용 시
     - 소득 무관 전액 지원

 11) 유아학비 (유치원)
     - 만 3~5세 유치원 이용

 12) 아이돌봄 서비스
     - 만 12세 이하 / 가정 돌봄 인력 파견
     - 소득별 차등 본인부담

 13) 육아휴직급여
     - 통상임금 80% 지급 (고용보험)
     - 만 8세 또는 초2 이하

 14) 육아기 단축근무·10시 출근제 (사업주 지원)
     - 자녀 만 12세 이하

[B] 자동 동기화 대상 API
==============================================================================

쑥 마을은 정책이 자주 바뀐다는 점을 가장 큰 문제로 보고, 가능한 한 많은 정책을
공식 API로 자동 동기화하는 걸 목표로 합니다.

------------------------------------------------------------------------------
B-1. 행정안전부 — 대한민국 공공서비스(혜택) 정보 API
------------------------------------------------------------------------------
  · 출처: 공공데이터포털 (data.go.kr)
  · URL: https://www.data.go.kr/data/15113968/openapi.do
  · 별칭: 정부24 보조금24
  · 제공 정보: 정부·지자체·공공기관·교육청의 수혜서비스 목록과 상세
  · 키 발급: data.go.kr 회원가입 → 활용 신청 (보통 자동승인)
  · 형식: REST + JSON
  · 정책 매칭: 대상 연령·소득·지역·자격 필드 포함되어 우리 모델과 매핑 좋음
  · 우선순위: ★★★ 가장 핵심 — Phase 1에서 통합 목표
  · 예상 커버리지: 중앙정부 정책 ~80% + 지자체 정책 일부

------------------------------------------------------------------------------
B-2. 보건복지부 — 보건복지 공공데이터 / 복지로 연동
------------------------------------------------------------------------------
  · 출처: https://www.bokjiro.go.kr (복지로)
  · 보건복지부 공공데이터: https://www.mohw.go.kr/menu.es?mid=a10104010100
  · 제공 정보: 복지 사업 상세 (산모·신생아 건강관리, 부모급여 등)
  · 키 발급: data.go.kr 또는 보건복지부 별도
  · 정책 매칭: 보육·복지 카테고리 보강용
  · 우선순위: ★★

------------------------------------------------------------------------------
B-3. 여성가족부 — 정책 자료 API
------------------------------------------------------------------------------
  · 출처: 공공데이터포털
  · URL: https://www.data.go.kr/dataset/15027362/openapi.do
  · 제공 정보: 한부모가족·양성평등·여성 정책
  · 정책 매칭: 한부모·여성 한정 정책 보완
  · 우선순위: ★★

------------------------------------------------------------------------------
B-4. 서울특별시 — 열린데이터광장
------------------------------------------------------------------------------
  · 출처: https://data.seoul.go.kr
  · 제공 정보: 서울시 임신·출산·육아 관련 데이터셋 다수
  · 임신·출산 정보센터: https://seoul-agi.seoul.go.kr
  · 탄생육아 몽땅정보통: https://umppa.seoul.go.kr
  · 키 발급: 회원가입 후 인증키 발급
  · 정책 매칭: 서울특별시 정책 + 일부 자치구 정책
  · 우선순위: ★★★ (서울 우선 출시 전략에 부합)

------------------------------------------------------------------------------
B-5. 자치구 — 자치구별 열린데이터광장 (선택적)
------------------------------------------------------------------------------
  · 강남구: https://data.gangnam.go.kr (확인 필요)
  · 서초구: https://data.seocho.go.kr
  · 송파구, 마포구, 성동구: 자치구 홈페이지 / 데이터광장
  · 자치구 정책은 API 표준화 정도가 들쭉날쭉 — 1차는 수동 등록 권장,
    2차에서 자동화 검토
  · 우선순위: ★ (자동화 우선순위 낮음, 수동 등록으로 시작)
