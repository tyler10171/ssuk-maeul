## 핵심 결정 요약

- **MVP 범위**: 탐색 + 조건 매칭 + 이메일/웹푸시 알림 + 자동 동기화 + 관리자
- **정책 범위**: 중앙정부 + 17개 광역 + 시군구 (서울 5개 구 우선)
- **알림 채널**: 이메일 (Resend) + 웹 푸시 + 사이트 내 알림함
- **기술 스택**: Next.js 15 + Supabase + Vercel + Resend + Tailwind
- **인증**: 이메일 + 비밀번호 (Phase 2에서 카카오/구글 추가)
- **데이터 모델**: User · Child · Policy · Match · Notification (5개 엔티티)
